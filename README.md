## How To Use :closed_lock_with_key:

```bash
# Clone this repository
$ git clone git@github.com:mux-mux/portfolio.git

# Go into the repository
$ cd portfolio

# Install dependencies
$ npm install

# Open browser & listen to scss changes
$ npm run dev
```
