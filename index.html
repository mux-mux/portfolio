Kanban

Driven by my goal to master JavaScript and DOM manipulation, I took on the challenge of building a fully functional drag-and-drop Kanban board with an integrated Pomodoro timer. To bring this project to life, I started by mapping out a clear schematic layout, carefully planning its structure and functionality. As I transformed my vision into working code, I encountered several complex challenges that tested my problem-solving abilities:

  • Efficiently removing event listeners (I debugged the code using Chrome DevTools' Event Listeners tab, refining my approach to ensure optimal performance)
  • Restoring Tab key focus after DOM re-renders (I assigned each task a unique ID and tracked focused elements dynamically)
  • Maintaining the Pomodoro timer state across DOM updates (I leveraged localStorage to persist the timer and session data, ensuring a seamless user experience)

Each challenge pushed me to think critically, refine my approach, and embrace debugging as a learning opportunity. Through persistence, humility, and a commitment to improving my craft, I built not just a useful tool but also a deeper understanding of JavaScript‚Äôs intricacies.
